# FleetFlow - Intelligent Fleet Management System

FleetFlow is a premium, futuristic, and highly responsive fleet management application designed to streamline logistics, track personnel, and optimize operational costs. Built with a modern tech stack and featuring a stunning glassmorphism UI, FleetFlow provides real-time insights for fleet managers, dispatchers, safety officers, and financial analysts.

## 🚀 Key Features

### 📊 Role-Based Dashboards
- **Fleet Manager**: Overview of total vehicles, active drivers, and critical maintenance alerts.
- **Dispatcher**: Real-time trip monitoring and rapid assignment control.
- **Safety Officer**: Driver safety scores, licensing compliance, and incident tracking.
- **Financial Analyst**: Breakdown of fuel costs, maintenance spend, and cost-per-vehicle metrics.

### 🚛 Vehicle & Driver Management
- **Vehicle Registry**: Track make, model, status, and odometer readings.
- **Driver Roster**: Manage personnel, certification expiry, and safety performance.
- **Compliance Tracking**: Automatic alerts for license expiry and maintenance needs.

### 🛣️ Operations & Logistics
- **Trip Manifest**: Dispatch control with origin/destination tracking and cargo weight management.
- **Fuel Ledger**: Detailed consumption tracking and spend analysis for entire fleet.
- **Maintenance Log**: Comprehensive service history and cost reporting.

### 🌗 Dynamic Theming
- **Premium Design**: State-of-the-art glassmorphism aesthetic.
- **Dark/Light Mode**: Seamlessly switch between themes with locally persisted preferences.

## 🛠️ Technology Stack

- **Frontend**: React.js, Vite, Tailwind CSS, Lucide React
- **State Management**: Custom data stores for real-time reactivity
- **Icons**: Lucide React
- **Theming**: CSS Variables with `data-theme` attribute
- **Layout**: Dynamic role-based sidebar and glassmorphic navigation

## 🏁 Getting Started

### Prerequisites
- Node.js (v16+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ParvSuhagiya/fleetWOrkFlow_oddo.git
   ```
2. Navigate to the project directory:
   ```bash
   cd fleetWOrkFlow_oddo
   ```
3. Install dependencies:
   ```bash
   # For Frontend
   cd frontend
   npm install

   # For Backend
   cd ../backend
   npm install
   ```

### Running the Application
1. Start the backend server:
   ```bash
   cd backend
   npm run dev
   ```
2. Start the frontend development server:
   ```bash
   cd ../frontend
   npm run dev
   ```
